# A&E Performance AB — Hemsida

Byggt med **React 18 + Vite + TanStack Router**.

## Kom igång

```bash
# 1. Installera beroenden
npm install

# 2. Starta dev-server (öppnas på http://localhost:5173)
npm run dev

# 3. Bygg för produktion
npm run build
```
